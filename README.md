# Gemini Clone

## Project Overview

Gemini Clone is a web application designed to provide users with a personalized assistant experience. The application features a user-friendly interface, enabling users to input prompts and receive contextual responses. It includes a sidebar for navigation and recent prompts, and a main area for inputting new prompts and viewing responses.

## Features

- **User Interface**
  - Clean and intuitive design
  - Sidebar navigation with recent prompts and actions
  - Main input area for prompt entry and response display

- **Functionality**
  - Typing effect for responses
  - Contextual responses to user prompts
  - Recent prompts saved for quick access
  - Dynamic loader while waiting for responses

## Technologies Used

- **Frontend:**
  - React.js
  - CSS for styling


## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/yourusername/gemini-clone.git
    cd gemini-clone
    ```

2. Install dependencies:

    ```sh
    npm install
    # or
    yarn install
    ```

### Running the Application

1. Start the development server:

    ```sh
    npm start
    # or
    yarn start
    ```

2. Open your browser and navigate to `http://localhost:3000`.

## Usage

1. Enter a prompt in the input box.
2. Click the send icon or press Enter to submit the prompt.
3. View the response generated by the application.
4. Use the sidebar to navigate recent prompts, start a new chat, or access settings.
